# test-uv

A short description of the project.

Python package `test_uv` (managed with `uv`).

## Installation

### Using `uv`

Create a virtual environment, install dependencies, and install the project:

```bash
uv sync
```

This creates `.venv/` and installs the project in editable mode.

### Run

```bash
uv run python -c "import test_uv; print('test-uv is installed')"
```
